# 🏖️ Sandbox Management

*This section is an exclusive tool visible only to System Admins ("God-Admins").*

If you are hosting FloraSync for multiple users or running a public demonstration, you might want a safe place for people to click buttons without ruining real garden data.

The Sandbox Management tool allows you to instantly wipe the entire "Demo Garden" workspace and restore it perfectly back to its initial seed state with a single click!

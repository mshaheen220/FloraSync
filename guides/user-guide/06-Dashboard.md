# The Command Center (Dashboard)

The Dashboard is a dynamic sorting engine that builds an "Attention Queue" to surface the most urgent tasks and relevant insights using a modular widget system.

Users can reorder or completely hide any of these widgets to tailor the dashboard to their specific workflow. Layout preferences sync to the individual user profile, so your layout won't affect your collaborators.

## Dashboard Widgets
Below is the hierarchy of widgets available on the dashboard. See their individual documentation files for more details.

* Attention Queue Logic
* Garden Vitality
* Quick Actions
* Urgent Location Care
* Needs Watering & Hungry Plants
* Health Watchlist
* Approaching Harvest
* The Nursery
* Random Spotlight & Fun Facts
* Garden Pulse
* Local Weather & Soil Insights

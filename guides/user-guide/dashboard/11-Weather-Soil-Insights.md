# Local Weather & Soil Insights (Opt-In Addon)

While FloraSync is built to be a fully local, offline-first application, it includes an official "Weather Widget" addon. Because this widget requires an active internet connection to fetch live data, it is intentionally deactivated by default to respect your privacy and local network preferences. Once activated by a Garden Owner or Admin, it uses the **Open-Meteo** API to provide hyper-local weather conditions, soil temperatures, evapotranspiration (water loss) rates, and impending storm alerts—translating raw meteorological data into actionable gardening advice right on your dashboard.
